Barbotan Paper

Original Image

Modified Image

Threshold values used: 120 and 255 in GIMP.
I don't think there is one single "best" threshold value because it's all relative to what the viewer prefers.
If you want to be able to read it more clearly with some spots of the background, you can go a little higher.
Similarly, if you want to completely eliminate the background while sacrificing readability, you can go lower.

Intensity Equality

Original Image

Modified Image

To do this I looked at the HSV values of the backgrounds using the color picker in GIMP and modified the inside color's V values (intensity) to match. Then, I desaturated the whole image to see if the colors disappeared and they did! Also I noticed that there was a little black dot in the original image on the top left box but wasn't sure if I should remove it so I just left it.

Sheet Music

Original Image

Modified Image

You can click on either one to view the full resolution of the image.
The enhanced image is much easier to read, but there are some minor flaws with it. First, when I was doing the edge detection, different values would sometimes make the filled in black notes into hollow notes. In music this changes it from a quarter note to a half note which is a big deal. Another flaw I found was that some tiny black dots appeared which are "noise" for this image. And finally, if you set the settings high enough to edge detection in gimp you can actually see hidden notes which might be from the other side of the sheet of paper.
