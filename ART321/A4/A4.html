Design a Net Art Piece

Lester's Public Drawing Board!

The idea that I was thinking of doing for my final project was to create a way for artists from all over the world brainstorm ideas and create art pieces on the same canvas. So this design assignment was a great opportunity to start thinking about it.

Some useful things that could be done with this project:

  • Great for making quick sketches for prototyping / storyboarding.
  • Great for brainstorming ideas
  • Can be turned into a game of some sort (like DrawMyThing)


How it’ll work (roughly)

2 Main Parts: Client and Server


Client

  • Loads the webpage and has to click start drawing to connect to the server. They can enter a username that sends a message to the server that registers that username on the server and lets them start drawing
  • Then when the user clicks on the webpage, it sends a message to the server with x and y coordinates on the page, username or id from the user, and a drawing/not drawing state. Maybe another thing that could be added is an RGB value for different colors.

Server

  • When it receives a new user message, it saves them on the server so other people can see their cursors.
  • When a user who is in the drawing mode clicks and sends their message, the server receives it and modifies the board by drawing a line at the x and y coordinates, their name, and drawing state from the client’s message.
  • Maybe I could add a chat feature in the future.
